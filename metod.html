Основні методи в JavaScript 1. Рядки (String) | Метод | Що робить | Приклад | | ------------------ | ----------------------------- | -------------------------------- | | `length` | Довжина рядка | `"Hello".length → 5` | | `toUpperCase()` | Всі літери великі | `"hi".toUpperCase() → "HI"` | | `toLowerCase()` | Всі літери маленькі | `"HI".toLowerCase() → "hi"` | | `trim()` | Вирізає пробіли спереду/ззаду | `" hi ".trim() → "hi"` | | `split(sep)` | Розбиває рядок на масив | `"a,b".split(",") → ["a","b"]` | | `includes(sub)` | Перевірка на наявність | `"abc".includes("b") → true` | | `replace(old,new)` | Замінює частину рядка | `"abc".replace("a","x") → "xbc"` | 2. Числа (Number / Math) | Метод | Що робить | Приклад | | ----------------- | ------------------------- | --------------------------- | | `Math.round(x)` | Округлення до найближчого | `Math.round(4.6) → 5` | | `Math.floor(x)` | Округлення вниз | `Math.floor(4.9) → 4` | | `Math.ceil(x)` | Округлення вверх | `Math.ceil(4.1) → 5` | | `Math.random()` | Випадкове число 0–1 | `Math.random() → 0.234` | | `parseInt(str)` | Рядок → ціле число | `parseInt("123") → 123` | | `parseFloat(str)` | Рядок → число з комою | `parseFloat("3.14") → 3.14` | 3. Масиви (Array) | Метод | Що робить | Приклад | | ------------------- | --------------------------- | ------------------------------------- | | `push(el)` | Додає в кінець | `[1,2].push(3) → [1,2,3]` | | `pop()` | Видаляє останній | `[1,2,3].pop() → [1,2]` | | `shift()` | Видаляє перший | `[1,2].shift() → [2]` | | `unshift(el)` | Додає на початок | `[2].unshift(1) → [1,2]` | | `slice(s,e)` | Копія частини масиву | `[1,2,3].slice(1,3) → [2,3]` | | `splice(s,d,...el)` | Видал/додати елементи | `[1,2,3].splice(1,1,"x") → [1,"x",3]` | | `forEach(fn)` | Виконує функцію для кожного | `[1,2].forEach(n=>console.log(n))` | | `map(fn)` | Новий масив після функції | `[1,2].map(n=>n*2) → [2,4]` | | `filter(fn)` | Новий масив по умові | `[1,2,3].filter(n>1) → [2,3]` | | `includes(v)` | Перевірка наявності | `[1,2,3].includes(2) → true` | | `indexOf(v)` | Індекс елемента | `[1,2,3].indexOf(2) → 1` | 4. Об’єкти (Object) | Метод | Що робить | Приклад | | --------------------- | ------------------ | ------------------------------------ | | `Object.keys(obj)` | Масив ключів | `Object.keys({a:1}) → ["a"]` | | `Object.values(obj)` | Масив значень | `Object.values({a:1}) → [1]` | | `Object.entries(obj)` | Масив \[ключ,знач] | `Object.entries({a:1}) → [["a",1]]` | | `hasOwnProperty(k)` | Чи є ключ | `({a:1}).hasOwnProperty("a") → true` | 5. Інші | Метод | Що робить | Приклад | | --------------------- | --------------------- | ----------------------------------------- | | `setTimeout(fn,ms)` | Виконати через час | `setTimeout(()=>console.log("Hi"),1000)` | | `setInterval(fn,ms)` | Виконувати кожні N мс | `setInterval(()=>console.log("Hi"),1000)` | | `JSON.stringify(obj)` | Об’єкт → рядок | `JSON.stringify({a:1}) → '{"a":1}'` | | `JSON.parse(str)` | Рядок → об’єкт | `JSON.parse('{"a":1}') → {a:1}` |
